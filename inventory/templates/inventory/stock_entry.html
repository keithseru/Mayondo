Instructions
  • Select Product Variant: Choose the item to update
  • Enter Quantity:
    • Positive numbers (e.g., 50) = Add stock
    • Negative numbers (e.g., -10) = Reduce stock
  • Entry Types:
    • Addition - New stock received (auto-adds)
    • Return - Customer returns (auto-adds)
    • Damage - Damaged goods (auto-deducts)
    • Adjustment - Manual correction
    • Correction - Inventory count fix
  • Notes: Add reason for entry (recommended)
Important
  • DAMAGE entries: System automatically deducts from stock, even if you enter a positive number
  • ADDITION/RETURN entries: System automatically adds to stock
  • Negative stock prevention: System will reject entries that would make stock negative
Examples

Scenario 1: Received new stock

  • Quantity: 100
  • Type: Stock Addition
  • Result: +100 units

Scenario 2: Found damaged items

  • Quantity: 5 (enter positive)
  • Type: Damaged Goods
  • Result: -5 units (auto-deducted)

Scenario 3: Manual reduction

  • Quantity: -8
  • Type: Stock Adjustment
  • Result: -8 units
Best Practices
  • Always add notes explaining the reason for stock changes
  • Use Damage type for broken/spoiled items
  • Use Return type for customer returns
  • Use Addition type when receiving new stock
  • Use Correction type for inventory count fixes
  • Double-check quantity before submitting
